## DOSBox Staging

DOSBox Staging is a modern continuation of DOSBox, a DOS-emulator for running classic DOS games and applications.

This snap is a community-maintained unofficial build of [DOSBox Staging](https://github.com/dosbox-staging/dosbox-staging).

[![Get it from the Snap Store](https://snapcraft.io/static/images/badges/en/snap-store-black.svg)](https://snapcraft.io/dosbox-staging)

## Install

Install on any [supported Linux](https://snapcraft.io/docs/installing-snapd) distribution:

```
snap install dosbox-staging
```
